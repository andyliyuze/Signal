# Signal

## 说明

一个基于Asp.Net 以及Redis实现的聊天系统


### 技术

#### 1.Redis，用作缓存，高并发处理。
#### 2.SignalR , 实时通讯框架。
#### 3. windows服务，Topshelf，quartz.net ，作来同步数据。

### 功能

#### 1.登录注册
#### 2.好友搜索，添加好友
#### 3.群搜索，群创建，添加
#### 3.单对单聊天，群聊天
